## 欢迎来到我的个人空间！

今天是一个特别的日子，我的个人博客终于正式搭建完毕并上线了。

作为一个热爱技术、崇尚终身学习的软件工程师，我一直希望能有一个完全属于自己的角落，用来沉淀自己的思考、总结技术经验、以及记录生活中的琐碎点滴。

### 为什么选择 Hugo + GitHub Pages？

在决定搭建博客时，我对比了许多方案（比如 WordPress、Notion、Jekyll、Hexo 等），最终我选择了 **Hugo** 配合 **GitHub Pages**。原因有以下几点：

1. **极致的渲染速度**：Hugo 是用 Go 语言编写的，其构建速度极其惊人。通常只需要几毫秒到几十毫秒，就能将成百上千篇 Markdown 文章渲染成静态页面。
2. **纯粹的 Markdown 写作体验**：我只需要用我最熟悉、最喜欢的 Markdown 格式编写文章，其他的渲染工作完全交由后台自动处理。
3. **完全免费的静态托管**：GitHub Pages 提供了稳定、快速且完全免费的全球 CDN 托管服务，让我可以把精力 100% 放在内容创作上。
4. **轻量与可控**：这次我使用了一套完全定制的、极简高端的 CSS 响应式布局，避免了引入第三方庞大主题所带来的代码臃肿和网络延迟。

### 接下来会写些什么？

在这个博客里，我计划在未来分享以下内容：
- **技术总结**：日常开发中踩过的坑、架构设计的思考、新技术的源码解读与实践。
- **效率工具**：能够极大提升生产力的工具软件、开发环境配置及工作流心得。
- **生活随记**：读书笔记、观影感悟、以及生活里有趣的片段。
